# Full-Stack Developer Portfolio & Admin Dashboard

A high-performance, responsive portfolio website built with the **MERN Stack** (MongoDB, Express, React, Node.js). This project features a live public portfolio for showcasing work and a private, password-protected administrative panel for real-time data management.

## 🚀 Live Links
- **Public Portfolio:** (https://portfolio-website-iota-ten-88.vercel.app)
- **Backend API:** (https://portfolio-backend-cq7w.onrender.com)

## 🛠️ Tech Stack
- **Frontend:** React.js, Vite, Tailwind CSS, GSAP (Animations), Framer Motion
- **Backend:** Node.js, Express.js
- **Database:** MongoDB Atlas (Mongoose ODM)
- **Cloud Storage:** Cloudinary (Project Images & Certifications)
- **Deployment:** Vercel (Frontend), Render (Backend)

## ✨ Key Features
- **Dynamic Content:** All projects, skills, and honors are fetched from a live MongoDB database.
- **Admin CMS:** A dedicated internal dashboard to Create, Read, Update, and Delete (CRUD) data without touching code.
- **Smooth UX:** Advanced scroll animations using GSAP and responsive layouts for all devices.
- **Secure Contact Form:** Integrated validation and real-time messaging.
